## 📚 Documentation

### Setup

1. **Clone the repository:**

   ```sh
   git clone https://github.com/cojocaru-david/cojocarudavid.me.git
   ```

2. **Install dependencies:**

   ```sh
   cd cojocarudavid.me
   npm install
   ```

3. **Run the development server:**

   ```sh
   npm run dev
   ```

### Usage

- **Build for production:**

  ```sh
  npm run build
  ```

- **Deploy:**

  ```sh
  npm run deploy
  ```
